# BASKETBALL

Interactive basketball court to get easily the statistics of a player compared to his league.

## Init

`npm install`

Then

`npm start`

open your localhost on :

`http://localhost:8080/`

That's it :)

## TECH STACK

- ReactJS
- ES6/7
- Webpack
- PostCSS
